# TechFlow Solutions - Task Manager

## Descrição e Objetivo
Sistema de gerenciamento de tarefas desenvolvido sob demanda para uma startup de logística fictícia. O objetivo principal é fornecer uma plataforma que permita acompanhar o fluxo de trabalho em tempo real, priorizar tarefas críticas e monitorar o desempenho da equipe.

## Metodologia Adotada
O desenvolvimento segue metodologias ágeis, utilizando a estrutura Kanban gerida através do GitHub Projects para o controle visual do fluxo de entregas (To Do, In Progress, Done).
